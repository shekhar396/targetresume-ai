# Vision

TargetResume AI is designed as a focused AI productivity product for job seekers who need to tailor their resume quickly without starting from a blank page.

The core idea is simple: a candidate already has useful career context in their LinkedIn profile or professional summary. TargetResume AI uses that existing material, combines it with a target company and role, and produces a clean, role-aligned resume that can be reviewed, edited, and exported.

## Product Positioning

This project is not a generic resume builder. It is a targeted resume generation assistant built around a specific workflow:

1. Paste profile or LinkedIn content.
2. Define the company and role.
3. Optionally provide a job description.
4. Generate a tailored resume.
5. Preview and export a polished PDF.

The product should feel credible, efficient, and practical. It should communicate the judgment expected from a real SaaS product: a clean interface, understandable outputs, secure handling of API credentials, and a clear boundary between AI assistance and user responsibility.

## Target Users

- Software engineers and technical professionals applying to specific roles.
- Career switchers who need help reframing existing experience.
- Students or early-career candidates who want a more structured resume.
- Portfolio reviewers and recruiters evaluating AI product development quality.

## v1.0 Success Criteria

The first release is successful if a user can:

- Provide professional profile text and target role information.
- Generate a coherent, tailored resume with role-relevant emphasis.
- View the resume in a clean HTML layout.
- Export the resume as a readable, professional PDF.
- Understand the product flow without onboarding or documentation.

## Product Principles

- Keep the core workflow fast and obvious.
- Treat AI output as a strong draft, not an unquestioned final authority.
- Prefer structured, predictable resume data over raw prose blobs.
- Keep implementation simple, but design with production concerns in mind.
- Build a portfolio project that shows product thinking as well as technical execution.
